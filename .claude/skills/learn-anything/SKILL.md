# Learn Anything — AI辅助快速学习法

## 目标

从门外汉到资深水平。不走线性路径，用树状展开：遇到不懂的直接问，含糊的直接展开，直到收敛。

## 工作流

### 第一阶段：选书

用户提供 TOPIC，你完成：

1. 找出该领域工业界公认最好的 **5 本书**（附简介、适读人群、豆瓣/GoodReads 评分）
2. 请用户确认或替换书单

### 第二阶段：生成合成书

根据 5 本书内容，合成一本新书：

- 目录列出所有章节，每章标注主要来源书籍
- 每章结构固定（见下方模板）
- 代码示例优先使用用户指定的真实项目代码，要注明来源和写注释

**章节模板**：

```
## 第N章：章节标题

### 1. 通识与知识讲解
- 每个知识点注明来源：《书名》第X章
- 概念要与其他知识点建立关联

### 2. 重点辅导
- 用代码示例说明核心机制（注明来源或标注"AI生成"）
- 用图表增强理解（注明来源或标注"AI生成"）

### 3. 习题
- 5~10 道题，保证做完后达到资深水平
- 包含填空、简答、编程、设计题混合
```

### 第三阶段：章节学习循环

每章按以下节奏迭代，直到"拿捏"：

```
[ ] 通读本章
[ ] 遇到不懂的知识点 → 直接展开讨论，不用管循序渐进
[ ] 讨论收敛后 → 更新到对应章节，附引用
[ ] 完成习题
[ ] 提交答案 + 自己的理解 → 让 AI 批改讲解
[ ] 如果 AI 讲解有缺失或错误 → 指出，让 AI 更正并完善
[ ] 批改内容更新进书的章节
[ ] 本章拿捏 → 进入下一章
```

### 第四阶段：巩固记忆

全部章节完成后：每天从头快速过一遍，直到形成长久记忆。

## 交互规则

- **树状展开**：任何时候遇到不懂的，直接说"展开 XXX"，AI立即深入讲解
- **收敛确认**：展开讨论后，AI主动问"这个问题清楚了吗？继续章节？"
- **习题批改**：用户提交答案时，AI给出：评分 / 逐题讲解 / 补充用户提到的理解 / 纠正错误
- **章节更新**：每次讨论或批改后，AI将增量内容更新追加到对应章节末尾（标注日期）

## 扩展用途

- 快速准备考试：将 TOPIC 换成考试科目，习题换成历年真题风格
- 技术评审准备：将 TOPIC 换成面试方向
